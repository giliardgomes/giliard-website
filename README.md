# Giliard Gomes - Personal Portfolio

This repository contains the source code for my personal portfolio website, designed to showcase my case studies and work as a Product Designer. The site leverages a modern React architecture and integrates a headless CMS for dynamic, easily maintainable content.

## 🚀 Tech Stack

### Core
* **[Next.js](https://nextjs.org/) (v16):** The core React framework handling routing, server-side rendering, and static generation.
* **[React](https://react.dev/) (v19):** UI library for building component-driven interfaces.
* **[TypeScript](https://www.typescriptlang.org/):** Ensures type safety and improves the developer experience.

### Styling & Motion
* **[Styled Components](https://styled-components.com/):** CSS-in-JS library for scoped, component-level styling.
* **[Framer Motion](https://www.framer.com/motion/):** Powers the interactive motion design and fluid page transitions.
* **[Lucide React](https://lucide.dev/):** Clean and consistent SVG iconography.
* **React Syntax Highlighter:** Formats and highlights code snippets within technical case studies.

### Content Management (Headless CMS)
* **[Sanity](https://www.sanity.io/):** The backend content platform used to manage portfolio projects, text, and assets.
* **Portable Text:** Used to dynamically render rich text content from the Sanity Studio directly into React components.

### Utilities & Infrastructure
* **[Resend](https://resend.com/):** Email API client managing the website's contact form submissions.
* **Vercel Analytics & Speed Insights:** Built-in tracking for visitor metrics and Core Web Vitals.
